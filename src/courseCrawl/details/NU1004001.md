##  課程簡介Course Description
  1. 認識全球運動經紀產業與趨勢
  2. 實務操作-模擬真實情境學習經紀人必備能力
  3. 深入探討臺灣職業運動產業發展現況/危機處理策略
  4. 職業球員推薦檔案實作
  5. 模擬經紀團隊分組提案
