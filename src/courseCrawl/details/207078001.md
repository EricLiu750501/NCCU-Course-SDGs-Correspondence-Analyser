##  課程簡介Course Description
```
本課程介紹航空攝影測量的基本原理和應用，包括：如何利用航空數值攝影測量建立2D和3D模型，其背後的數學基礎和概念，工作流程和分析。除理論外亦輔以實習課程，讓學生實際操作航測軟體。
```
