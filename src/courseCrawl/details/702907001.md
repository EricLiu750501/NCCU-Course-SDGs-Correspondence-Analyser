##  課程簡介Course Description
本課程將介紹各種不同的睡眠疾患，並透過基本睡眠機制的瞭解，能進一步瞭解睡眠疾患的成因，以及評估與處遇方式。此外，課程亦將探討專為研究睡眠障礙所建立的多種動物模型，並說明如何有效運用這些模型以探討睡眠障礙與其他疾病之間的關聯。課程內容涵蓋基礎與臨床層面，旨在全面提升學生對睡眠相關議題的理解與研究能力。

##  課程目標與學習成效Course Objectives & Learning Outcomes 
  1. 瞭解睡眠調節的基本機制。
  2. 瞭解睡眠疾患的定義、症狀與診斷標準。
  3. 瞭解睡眠疾患的病因、評估及處遇。
  4. 瞭解睡眠疾患的心理及行為因子。
  5. 瞭解動物模型在睡眠疾患研究中的應用。

##  評量工具與策略、評分標準成效Evaluation Criteria
  1. 上課參與（15％）：同學需準時上下課，並參與課堂討論。
  2. 課堂報告（20％）：同學需於課堂上針對特定睡眠主題或睡眠疾患進行報告。
  3. 課堂小考（20％）：針對上課內容進行課堂小考以評估同學的了解程度。
  4. 家庭作業（15％）：同學需針對特定主題進行觀察記錄及反思，於課堂中分享討論，並繳交家庭作業。
  5. 期末報告（30％）：包含口頭及書面報告，分組執行睡眠行為改變研究，以自我觀察或小規模收案的方式，觀察及測量睡眠行為改變的影響。
