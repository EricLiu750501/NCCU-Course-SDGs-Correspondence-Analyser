##  課程簡介Course Description
讓同學了解國際組織的種類，認識各種不同的國際組織及其發展，並討論土耳其共和國在各國際組織間所扮演之角色。

  3. 學期內病假次數不得超過2次；事假次數不得超過1次，超過者仍以缺席論。 
  4. 教師點名後才到教室者，不得要求老師補點名。 
  5. 總課堂數缺席四堂者，學期總成績不得超過80；總課堂數缺席五堂者，學期總成績不得超過70；總課堂數缺席六堂者，學期總成績不得超過60。
