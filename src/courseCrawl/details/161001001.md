##  課程簡介Course Description
本課程介紹漢語語言學基本知識，包括語音、句法、構詞、語意、語用和社會語言學等。

##  評量工具與策略、評分標準成效Evaluation Criteria
1. 課堂出席與討論參與20% 
2. 期中考20%
3. 期末考20%
4. 期末口頭報告20% 
針對一個漢語語言學主題、現象或結構（例如：三聲變調、把字句、存現句、語碼轉換）提出語言學上的解釋，整理現有文獻即可。
5. 期末書面報告20% 
期末口頭報告的書面版本，並統整於口頭報告時得到的回饋。請準時繳交，**逾期不收** 。
