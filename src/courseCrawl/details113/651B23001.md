##  課程簡介Course Description
本課程為111學年以後入學的法律系博士班同學必修的論文研討課，對於碩士論文寫作也很有幫助，因此碩博合開。由於1學分的課程時間相當有限，本學期僅限6人選修，隔週上課，每次上課兩小時。 每學年上下學期都會開設，惟114學年第1學期因授課教師休假研究，將停開一次。雖然正式修課人數相當有限，不過非常歡迎碩、博士班同學到課堂旁聽，一同參與研討，精進研究技能，瞭解新法律領域與新議題。
在課程前半部，本學期以研討法釋義學、法政策學、比較法 與法律經濟分析 此四種研究方法，及其相互之間的互補及互動作為前半部課程重點。在此部分，每位修課同學須選擇一周時間，就指定之法釋義學或法經濟分析中文閱讀資料進行課堂報告，具體安排請見本教學大綱之每周課程進度與附掛檔案
有關課程後半部之論文研討，修課同學須有足以報告35分鐘之研究架構（ 例如學位論文大綱或單篇專題論文），以供在課堂提出報告，並進行研討。如係針對學位論文主題進行報告，由於課程時間有限，建議著重於論文整體架構而非特定章節內容，由教師及同學對論文架構進行討論思辨與經驗交流，依以往經驗對報告同學最有幫助。
上課時間：周四 19:00－21:00。上課日期為 02 / 20、03 / 13、03 / 27、04 / 10、04 / 24、05 / 08、05 / 22、06 / 05 及 06 / 19。其中05 / 22、06 / 05 及 06 / 19各由兩位同學報告研究架構，上課時間延長為 18:30 - 22:00。

##  課程目標與學習成效Course Objectives & Learning Outcomes 
本課程乃針對法學研究方法（含方法論取向與研究法）以及學位論文如何寫作所設計的論文研討課程。課程範圍除了個別領域之法律研究外，同時包含法律領域間或是法律與其他學科間的跨領域整合研究。為避免抽象講授或文獻研讀難以心領神會，且研究過程中所遭遇問題與克服技巧往往不易歸納為抽象建議或規則，本課程以實存之法學研究作為研討素材，由師生在課堂中分享仍在進行中之研究作品（work in progress），以做中學的方式，從實體與形式層面，共同探討可能隱含的問題與解決之道。

第一次：
課程主題：（1）課程簡介，說明上課方式及評分標準，瞭解修課同學背景。（2）法學研究方法
課程內容與指定閱讀：介紹常見的法學研究方法，包括法釋義學、法政策學與比較法，可參考最下方參考書目所列各種文獻。
教學活動與作業：教師講授，修課同學共同研討
學習投入時數：課堂時數2小時；課前/後學習時數3小時
第二～六次：
課程主題：法經濟分析與法釋義學
課程內容：分五周研讀法釋義學與法律經濟分析指定教材，每周由一位同學進行40分鐘 左右的課堂報告，其後由老師及同學進行研討交流
指定閱讀：張永健，法經濟分析：方法論20講，北京大學出版社，2023年7月1版（簡體中文書）
教學活動與作業：依進度事先閱讀指定文獻、課堂報告及研討。詳細進度分配請見附掛檔案。
學習投入時數：課堂時數2小時；課前/後學習時數3小時
第七～九次：
課程主題：論文寫作研討。
課程內容與指定閱讀：每次由兩位同學就研究架構進行課堂報告，長度各35分鐘。接著開放在場師生提問，各自提出意見，共同進行研討。每位同學報告及研討時間共約90分鐘。
教學活動與作業：修課同學應製作投影簡報，在上課前兩天（周三）中午以前提供予課程助理及教師 ，以便上傳Moodle平台供提問同學事先準備。
學習投入時數：課堂時數2小時；課前/後學習時數3小時
