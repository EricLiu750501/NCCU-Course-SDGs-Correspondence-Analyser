##  課程簡介Course Description
本課程聚焦於學校組織行為的理論與實證研究，旨在探討學校組織行為的核心概念、理論框架及其應用，包括領導行為、組織文化、決策過程、變革管理及組織學習等議題。授課重點以EAQ期刊中的相關文獻，結合研究方法(量化分析為主)，深化對教育組織行為的學術與應用理解。

##  課程目標與學習成效Course Objectives & Learning Outcomes 
#### 課程目標
  1. 探討學校組織行為的主要理論與實證研究成果。
  2. 培養學生批判性閱讀與分析教育組織行為研究文獻的能力。
  3. 引導學生以學術角度分析和解決教育組織問題。
  4. 提升學生對學校組織行為議題的研究能力與專業素養。
#### 學習成效
  1. 清楚說明學校組織行為的核心理論與概念。
  2. 批判性分析學術期刊文章，並提出研究改進建議。
  3. 結合理論與實證，針對實際教育問題提出解決策略。
  4. 發展與撰寫一篇具學術價值的組織行為相關文獻評論或研究。
