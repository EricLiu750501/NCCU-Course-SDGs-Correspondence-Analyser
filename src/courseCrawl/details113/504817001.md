##  課程簡介Course Description
（一）本課程為二年級選修課程， 延伸中級俄語、語法及會話課程內容，提升俄語口語表達及溝通能力。
（二）課程難度為俄語檢定考試一級（B1）。

課程主題與內容
1. 用餐：在餐廳、在速食店、請人推薦餐廳、點餐、付帳
2. 觀光：博物館購票與參觀、教堂參觀、欣賞表演、購買遊覽行程
3. 購物：超市、百貨、服裝店、攤販
4. 日常生活：在郵局、在銀行、使用提款機、在美髮店
5. 尋求協助：在路上、在醫院、在警局、在維修中心
教學活動與作業
1. 主題詞彙學習
2. 句型練習
3. 口語表達練習
4. 情境對話
