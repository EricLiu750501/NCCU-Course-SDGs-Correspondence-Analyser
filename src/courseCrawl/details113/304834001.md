##  課程簡介Course Description
This course will utilize the statistical software R to explore the uncertainty of random variables through programming. Students will apply learned statistical methods by simulating random variables. The course will primarily be conducted through lectures and real-time programming exercises.

##  課程目標與學習成效Course Objectives & Learning Outcomes 
This course aims to train students with profound knowledge using simulation to understand and learn a model. Develop students' intuition for dealing with statistical problems and their ability to implement practical ideas into programming languages.
