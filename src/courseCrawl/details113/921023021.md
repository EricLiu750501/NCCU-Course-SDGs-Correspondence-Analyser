##  課程簡介Course Description
本課程主要教導同學如何利用經濟學的分析工具與概念，瞭解如何運用最有效率的方法面對資源的有限性，進而達到管理目標的社會科學；並希望可以推廣運用至個人、家戶、政府、以及企業等各領域的決策。本課程包括了兩個部分：第一部分的授課內容由羅光達老師（財政學系）負責授課，而第二部分的授課內容則由王信實老師（經濟學系）負責授課。兩位老師的課程雖然獨立進行與評量，但課程內容的安排與進度仍是具有連貫性。

##  課程目標與學習成效Course Objectives & Learning Outcomes 
# 課程介紹：
經濟學主要在處理稀少資源進行最有效配置的問題。而管理經濟學，則在協助決策者（管理者）有效率地配置稀少資源、規劃企業策略、以及執行有效的策略。本部分課程主要先帶領各位同學認識經濟學的基本概念，從消費者與生產者，延伸討論到市場結構與政府角色等相關議題。透過本課程的學習，希望同學能夠活用經濟學的原理與分析方法，並進一步能將經濟學的方法，運用到管理決策相關的議題。
# 課程目標：
在本課程結束時，同學們應該能夠：
  1. 瞭解經濟學的基本分析方法與工具
  2. 認識供需市場與市場結構
  3. 學習以經濟學的思考方式來分析管理決策議題
  4. 學習參與對公共政策的討論
