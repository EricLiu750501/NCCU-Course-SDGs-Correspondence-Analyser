##  課程簡介Course Description
本課程為「 非資訊系之網頁設計開發入門課程」 ，歡迎對網站前端開發感興趣的同學一起學習，本堂課將以 Next.js 前端框架操作學習為主、網站系統理論為輔，同時搭配 Three.js (R3F) / Motion / Tailwind CSS 等網站 3D 與動態效果工具的學習，期待數位內容學程的同學能掌握實用的工具和技能，並將其應用於未來的工作和學習中，具備初階網站前端工程師與特效開發的能力。
備註１： 每次上課時請自備筆電，並充飽電  
備註２: 只要不會害怕程式碼，無經驗的同學都非常適合與歡迎，期待你也會有不錯的收穫  
備註３: 太多同學寄信詢問，本課程 限定數位內容學程大學部 學生選修，若要旁聽請第一週來教室會說明相關規定
