##  課程簡介Course Description
本課程無加簽  
若無修過統計學或排斥學習程式可能負擔較大，請謹慎考慮
This course provides an introduction to the fundamental concepts and widely used methods in Business Data Analytics, with a main focus on implementation using the R programming language. Key topics include data visualization, statistical testing, regression models, time-series forecasting, and machine learning. The course is designed to equip students with essential skills in data analysis and practical applications.

##  課程目標與學習成效Course Objectives & Learning Outcomes 
By the end of this course, students will be able to:
a) select and apply appropriate quantitative methods for analyzing various datasets,  
b) utilize the R programming language to analyze real-world data,  
c) create effective visualizations to communicate data insights, and  
d) deliver clear and compelling presentations to academic or professional audiences.
