##  課程簡介Course Description
Este curso está diseñado para familiarizar a los estudiantes con el formato y las diferentes pruebas del examen DELE. El curso está dividido en dos partes. En la primera se practicarán modelos de examen del nivel B1, y en la segunda parte, del nivel B2.
En las clases se verán y practicarán ejemplos de tareas de las cuatro pruebas del examen: comprensión auditiva, comprensión lectora, expresión e interacción escrita, y expresión e interacción oral. Las diferentes tareas de cada prueba tienen características especiales. En este curso se pondrá especial énfasis en que los estudiantes entiendan lo que se pide de ellos en cada tarea. Así mismo se propondrán estrategias para completar las diferentes tareas.

##  課程目標與學習成效Course Objectives & Learning Outcomes 
Al finalizar este curso, los estudiantes serán capaces de:
  * Explicar el formato del examen DELE B1 y B2
  * Explicar el tipo de preguntas y tareas de las diferentes pruebas del examen.
  * Entender las instrucciones específicas de todos los tipos de tareas.
  * Aplicar estrategias para completar las tareas más difíciles
  * Ponderar por sí mismos su nivel propio de español
