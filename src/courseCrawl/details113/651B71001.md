##  課程簡介Course Description
1. 最近的數十年，刑事法領域的立法相當頻繁。然而大部分的立法是否有在立法前做充分的立法上準備，則有疑問。
2. 關於立法的準備，即是在立法論上需以我國的現狀為前提，探討是否需要新的刑事處罰規定。在這樣的基礎上：
（1）對我國學說現況的清楚掌握；
（2）對我國實務現況的清楚掌握；
就是立法論的基本前提。無法掌握我國的現況，在我國頻繁參照外國法的情況下，參照外國法的立法將成為單純的引進外國法，使我國刑事法學成為外國法的殖民地而喪失主體性。另一方面，不參照外國法而在不清楚我國狀況下立法，也可能成為土法煉鋼。兩者不論何者，都可能帶來災難性的法律適用狀態。並導致之後必須要於短時間內再修法，引起不斷修法的惡性循環。
3. 本課程在內容上，關注的即是我國刑事法的狀態。但考慮到課程時間與修課同學所能支配的時間均有限，因此將會以我國學說的議論狀態的掌握為本課程的內容。
4. 在本課程中，將會提供不同主題的文獻清單給修課同學，經同學選擇並排定報告週次順序後，每週由修課同學負責介紹該主題的論文。
5. 介紹論文後，即從理論的觀點加以討論。
