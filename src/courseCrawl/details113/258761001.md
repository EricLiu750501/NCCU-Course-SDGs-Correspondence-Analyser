##  課程簡介Course Description
This is a 3-hour course designed for graduate students in the Department of Economics. The aim of this course is to provide fundamental knowledge and quantitative tools necessary for addressing macroeconomic issues. Additionally, I will briefly introduce how to apply dynamic programming techniques to handle basic macro models. We will engage in quantitative work based on economic models and conduct a few calibration/simulation exercises. We will "intensively" use Matlab software and Dynare program to conduct quantitative analysis. 
選課前請注意: 本課程主要探討總體經濟相關議題以及量化分析工具，不會涉及到計量方法或個體資料的使用。
