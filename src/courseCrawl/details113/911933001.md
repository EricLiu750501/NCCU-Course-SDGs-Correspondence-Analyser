##  課程簡介Course Description
```
本課程著重於數位科技賦能教育創新與管理研究及其應用。近年來，數位賦能與教育科技管理已成為全球教育的重要發展趨勢與教育政策管理關注的重要領域。數位科技的進步催生了新的教育典範模式，對於教育界而言，如何運用數位教育科技進行高效管理和有效實施的需求也變得更加迫切。本課程為學生提供探索這些國際間數位賦能教育上的重要趨勢並分享經驗和知識以改善教育科技管理實踐的機會，促進我國教育品質與教育創新實踐成效。
本課程將幫助學生學習各種模式的數位學習以及數位科技在教學/學習過程中可以發揮的作用，以及如何在課堂上參與這些過程。學生將熟練當今學校使用的一些數位工具，包括美國MIT麻省理工學院的系統科學(System Dynamics)建模和基於模擬的場景規劃，以及AI人工智慧應用教育議題。
本課程也將引導學生學習如何在教育環境中進行小型教育科技管理研究計畫​​。將教育實踐案例研究和結合PBL專案導向學習的永續發展教育和組織學習，以便在開展自己的教育科技管理研究專案之前為學生提供實用工具和應用研究策略。
```

##  課程目標與學習成效Course Objectives & Learning Outcomes 
```
本課程旨在協助學生深化對數位賦能和教育科技管理的理解，並學習如何管理教育科技以實現更好的教育品質與教學實踐。本課程的目標是讓學生：
1.了解數位賦能原理和教育科技趨勢以及教育創新議題。
2.管理教學和課程發展以及相關教育科技。
3.管理數位學習系統以及相關教育科技。
4.為組織學習制定想法、步驟並應用數位教育創新。
本課程學生有三個學習成果要求。首先，積極參與課堂討論。其次，每位學生必須準備一篇文獻評論（或案例評論）並向全班報告分享文獻/案例(包含對理論、方法和發現的分析)。第三，需要進行與教育科技和永續發展教育應用相關的案例研究。
```
