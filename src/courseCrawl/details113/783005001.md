##  課程簡介Course Description
本課程介紹機器學習的基本觀念與應用，包含監督式學習、無監督式學習及相關應用，課程中也會介紹機器學習的工具與套件(如 scikit-learn 等)，透過小組競賽或專題實作，學習使用開放資料(如 Open Data)、數據平台資料(如 Kaggle) 等，將機器學習技術應用於解決不同場域的問題。
* 課程範例以 Python 為主，建議同學具備 Python 程式設計能力或至少學習過一種程式語言，如 C/C++、R、Java 等。

##  課程目標與學習成效Course Objectives & Learning Outcomes 
了解機器學習的基本觀念與實務應用，透過小組競賽或專題實作，結合各自領域專業背景，使用開放資料(如 Open Data)、數據平台資料(如 Kaggle)等，將機器學習應用於解決不同場域的問題。
