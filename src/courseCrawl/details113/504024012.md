##  課程簡介Course Description
本課程希望能藉由語法規則的解說及練習使同學掌握詞法的基本知識和初步的句法概念，以期在言語交際中正確運用。
其主要教授內容：
1. 名詞、形容詞等單複數的第六格之構成與應用
2. 名詞、形容詞等單複數的第四格之構成與應用
3. 表達走、乘車之未綴運動動詞-1
4. 動詞的體
5. 名詞單數第三格之構成與應用

## 評量工具與策略
1. 出席（5 %）：上課是學生應盡的責任與義務，也是好的學習態度的基礎。若三次不到，出席成績將以零分計算。
2. 課堂表現（10%）：上課中我們除了講授課程以外，大部分採用刺激反應式的教學，希望透過該方式理解學生當下的學習狀況，以進行調整。學生上課前必須充分的複習及預習。
3. 作業 （10%）： 寫作業的意義，主要在於協助自己的學習，同時也在自我檢視學習的成果。透過作業的撰寫，能強化對於詞彙及語法使用的認知與能力。
4. 小考測驗（15%）：小考測驗不僅可以督促學生學習，亦可讓老師或學生了解學習成效，且得以調整教學方法或讓學生自我加強弱處。
5. 期中考（30%）: 學生學習成效之重要參考指標。
