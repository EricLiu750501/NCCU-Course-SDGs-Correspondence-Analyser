##  課程簡介Course Description
This course introduces machine learning techniques and their application in asset allocation. A strong background in calculus and linear algebra is required to comprehend the underlying properties of each method. The R programming language is the only primary tool to implement the machine learning algorithms.

##  課程目標與學習成效Course Objectives & Learning Outcomes 
  1. Gain knowledge on how to use machine learning and financial econometric models.
  2. Understand the limitations of each model
  3. Apply the statistical tools to solve financial problems.
  4. Gain the ability to critically evaluate the financial applications of machine learning and econometrics.
