##  課程簡介Course Description
本課程旨在以討論與引導的方式讓學生學習並且了解客家的語言以及文化，由語音、語言、語義以及文化不同主題進行教學，透過數位學習以及文獻閱讀，達到每週課程安排的學習目標。課程分為兩大部分：（一）介紹臺灣客語基本語言結構特性（含語音、構詞及句法語意），探討臺灣之華語、閩南語及客語接觸下，語言與文化之互動及對彼此的影響；（二）介紹臺灣客語特徵詞及文化意涵，透過不同文化題材的介紹，讓同學認識臺灣的客語及文化，培養同學對多元文化的尊重。

##  課程目標與學習成效Course Objectives & Learning Outcomes 
  1. 臺灣客語的語言結構特性，含語音、詞彙及語法
  2. 臺灣語言政策及語言接觸及其帶來的影響
  3. 臺灣客語特徵詞及其文化意涵
  4. 臺灣客家文化課題
  5. 賞析客語故事、客語俗語諺語、客語練習
學習成效
  1. 能知曉客家次方言的種類與分布
  2. 能知曉客家話語音、詞彙及語法之特色
  3. 認識語料庫之重要性及實作
  4. 認識客家特徵詞及客家文化意涵
  5. 具備文獻整理、組織與邏輯思考的能力
  6. 具備客語口說及書寫的基本能力
