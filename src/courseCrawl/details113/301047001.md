##  課程簡介Course Description
The one-semester course aims to introduce advanced undergraduate level students some well-established econometric methods with Python, a free and powerful programming language for scientific data analysis.
We will start with an introduction to Python, including how to install and update it, how to input, output and manipulate data, and how to use basic operations, functions and programming statements, etc. We will then focus on how to estimate and conduct statistical inferences of linear regression and some basic time series models with Python. The learned techniques will be applied to economic and financial data analyses.
The course will cover the following topics: 
An introduction to Python  
The simple regression model   
Multiple regression analysis: estimation   
Multiple regression analysis: inference   
Basic regression analysis with time-series data  
Time series models and forecasting

##  課程目標與學習成效Course Objectives & Learning Outcomes 
After learning the course, students are expected to know basic knowledge about econometric and statistical techniques and how to use them with Python in empirical analyses, which will be extremely useful for their future careers in the industry or study related to economics and finance.
