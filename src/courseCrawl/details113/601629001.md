##  課程簡介Course Description
本課程以刑事第一審程序之證據法則為重點。我國刑事司法實務往昔並不重視刑事證據法則，從上世紀80年代後，刑事司法實務界才開始有專門針對刑事證據法則爭議的判決見解出現。再加上1999年全國司法改革會議結論要啟動刑事司法改革，因而刑事訴訟法在歷經數次重大修法後，引進一些國外刑事訴訟與證據制度，其中，例如傳聞法則、證據禁止、交互詰問，鑑定等新增制度，皆涉及基本證據法則的改革。這些規定並不易理解。惟由於授課時間及客觀條件限制，本課程只能側重法律規定、學說及實務見解之講授，較為技術層面之實務演練，無法於課堂上操作。

##  課程目標與學習成效Course Objectives & Learning Outcomes 
本課程只介紹刑事程序之證據法則，不涉及民事或行政程序。
刑事證據法分為總論及各論兩部分：總論，處理刑事證據之共通法則；各論，處理不同類型之證據方法。
刑事證據法則不僅涉及學理，且須與實務結合。一方面，證據法則是刑事程序法（即刑事訴訟法）之一環。而刑事審判重心就在證據調查，證據調查是一連串證據法則與證據程序交互作用的結果，其複雜程度及對審判之重要性，遠非一般刑事程序規定所能比擬。另一方面，證據法則有其獨特的支配法則，尤其近年來修正通過之傳聞法則、證據禁止、交互詰問等規定，大大增加對證據法則理解之難度。
由於時間及客觀條件限制，本課程將置重於現行刑事訴訟法規定及實務見解，配合實例於課堂內講授，至於證據法則之實務演練，則無法於課堂內兼顧。
透過本課程之講授，學生能理解現行刑事訴訟法所規定之證據法則之理論與司法實務運作，並透過對國外制度的比較，對證據法則之比較法基礎也能有深入的認識。

課程大綱：
第一部份 總論
  * 證據法之意義與功能
  * 支配證據法之原則
    1. 證據法定原則vs.自由心證原則
    2. 直接審理原則vs.間接審理原則
    3. 言詞審理原則vs.書面審理原則
    4. 嚴格證明vs.自由證明
  * 證據法之實體法則與程序法則
    1. 實體法則
      1. 證據方法
      2. 證據能力
      3. 證據資料
    2. 程序法則
      1. 證據蒐集及提出──檢察官舉證責任、被告（辯護人）證據調查聲請權、法官職權調查證據義務、保全證據
      2. 調查程序
      3. 證據評價
  * 證據法之違法效果──證據使用禁止、證據排除法則（毒樹果實理論）
  * 私人取證之證據法則 
  * 證據保全（219之1以下）
第二部份 各論
  * 人證
    1. 基礎理論
    2. 調查程序－－交付詰問、對質
    3. 問題討論（1）：共同被告、共犯之自白？
    4. 問題討論（2）：傳聞法則
  * 鑑定
    1. 基礎理論
    2. 調查程序
    3. 問題討論（1）：檢查身體
    4. 問題討論（2）：機關鑑定
    5. 問題討論（3）：測謊？
  * 勘驗
    1. 基礎理論
    2. 調查程序
    3. 問題討論（1）：檢查身體
    4. 問題討論（2）：即時勘察
  * 文書證據
    1. 基礎理論
    2. 調查程序
    3. 問題討論（1）：警訊筆錄
    4. 問題討論（2）：傳聞法則──代替人證之文書證據（傳聞書證）
  * 被告陳述（自白）
    1. 基礎理論
    2. 調查程序──訊問、對質
    3. 問題討論（1）：補強證據（156第二項）
    4. 問題討論（2）：被告抗辯（刑求抗辯、幽靈抗辯等）

    1. 基礎理論－非獨立之證據方法
    2. 調查程序
