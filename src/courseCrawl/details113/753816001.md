##  課程簡介Course Description
```
本課程主要在介紹社群運算(Social Computing)相關技術與演算法，以培養同學應用在社群網絡相關議題的能力。
在授課方向上，將以Social Computing演算法設計為主，著重在學習思考與設計大量社群運算資料之有效率演算法。
先修課程：程式設計、線性代數、資料結構、演算法、資料庫系統、資料探勘
先備技術： C, Python, DBMS(Postgres, MySQL, or SQLserver), Github, Crawler等實作能力 
```
