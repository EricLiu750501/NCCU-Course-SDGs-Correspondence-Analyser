##  課程簡介Course Description
1. 本課程是以剛進入法律學系的學生為對象，就基礎法學的內容進行基本的說明，使修課同學能於課程中對於基礎法學作為法律學的學門具有如何的內容等問題，掌握其梗概。  
2. 本課程的內容將主要分為以下四個部分：  
(1)概說  
(2)法哲學思想概論  
(3)法與社會（法社會學）概論  
(4)法制史概論  
3. 本課程預定於期末進行期末考。
4. 課程大綱中的課程為預定課程，本課程實際上課內容的週次分配等，仍得由授課教師視實際教學需要調整。如有調整將再另行公告。
5. 本課程無法提供法律系（含雙主修）同學加簽。如果有法律系但需重修本課程的同學，要選修本課程，則以10人為限可加簽。
6. 如果是第一次選修本課程的同學，以教室容量能涵蓋的範圍內（約110人），可進行加簽。如果教室無法容納，則以30個名額為限提供加簽。如果需要加簽的人數超過30人，則由現場需要加簽的同學一起抽籤，抽出可加簽的30人的名單。

##  課程目標與學習成效Course Objectives & Learning Outcomes 
1. 透過本學期的學習，使學生能了解基礎法學所具有的主要學門  
2. 透過本學習的學習，使學生能了解法哲學思想相關的主要內容  
3. 透過本學期的學習，使學生能了解法與社會間的關係  
4. 透過本學期的學習，使學生能了解法制史對於法律學習的意義
