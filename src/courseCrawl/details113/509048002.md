##  課程簡介Course Description
Hierbei handelt es sich um einen Capstone-Kurses, in dem die Teilnehmenden ihre Leistungen erbringen sollen, die sie in dem gesamten Deutschstudium erworben haben. 
Eine Sitzung besteht aus zwei Teilen. Zuerst wird über ein aktuelles Thema gesprochen, indem die Nachrichtenbotin oder der -bote einen Artikel bzw. eine Textpassage aus deutschsprachigen Medien aussucht. Der Text soll am Samstag vor 12 Uhr bekannt geben werden, der dann im Unterricht erläutert und intepretiert wird. Anschließend wird in der Sitzung eine Textesorte anhand eines Beispiels besprochen. Siehe den Lehrplan. 
Eine weitere Aufgabe besteht darin, auf Deutsch über das National Taiwan Museum zu erzählen. Kursteilnehmende informieren sich durch ein Führung über das älteste Museum Taiwans im Allgemeinen. Zum Kurz wird dann eine Mitarbeiterin des Museums eingeladen, die regelmäßig Trainingsprogramme für eine englische Führung anbietet. Daraus sollen sich ein mündliche Referat und eine Hausarbeit ergeben.
