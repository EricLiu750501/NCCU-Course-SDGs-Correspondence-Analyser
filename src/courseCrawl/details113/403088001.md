##  課程簡介Course Description
姚經玉：透過電影專題、影展市場以及時事分析與觀察等方式，讓學生得以對華語電影產業、世界電影趨勢、女性議題電影、性別平權議題電影、電影政策及製作、市場行銷及影展實務等獲得深入理解。
薛聖棻：認識影視產業之生態現況（產業鏈與作業流程），本課程以相關產業實務重要操作面向來進行初探。輔以酌情邀請產業實務卓越人士，與學生直接對話，雙向建構產學效能。
課程分類：
(1)敘事能力檢視與培養。
(2)訊息蒐集的軌跡和有效元素的模式建立
(3)媒體趨勢與社群力銜接
(4)內容製作與閱聽行為改變之關連
(5)非戲劇類和戲劇類內容最新動態
(6)影視內容行銷的轉化與應變
(7)全球內容產業環境的比較與競合關係分析
(8)台灣的內容產業優勢與選擇

##  課程目標與學習成效Course Objectives & Learning Outcomes 
姚經玉：
  1. 透過專題研判及市場觀察，探討世界電影趨勢及市場。
  2. 透過電影實務，了解製作與發行在電影產業中的關鍵位置及其影響。
  3. 透過電影政策，分析當前台灣電影產業發展現況、結構及國際定位。
  4. 透過影展趨勢，分析女性及性別平權議題電影現況、國際影響及發展。
  5. 透過時事資訊觀察，了解台灣電影在全球電影的競爭優勢與劣勢。
薛聖棻：
  1. 課程內容隨產業生態變化，滾動式調整。
  2. 本課以互動式授課風格，增加學生實務感體驗。
  3. 多元內容分析，讓學生在學習過程中，思考未來發展方向。
  4. 內容行銷的觀念與能力建立。
  5. 以全球視角，組成有國際競爭力的團隊。
