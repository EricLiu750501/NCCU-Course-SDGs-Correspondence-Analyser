##  課程簡介Course Description
課程簡介：土地與建物的關係緊密，建築行為與建築物管理使用等涉及土地的合理使用。本課程授課內容第一部分將焦點置於「建築法」，目標在使同學理解個體管制與土地利用管制對建築行為與建築物的規範，並透過司法判決見解掌握實務上常見爭議。第二部分則著重在「公寓大廈管理條例」，使同學了解公寓大廈區分所有建築物的所有權及用益關係與管理，目標在釐清各相關人的權利義務關係及實務上常見糾紛背後的原因為何，以具備認識及處理不動產建築管理問題的能力。
  1. 建立對建築法規範核心內容的知識。
  2. 掌握對於公寓大廈管理條例及區分所有制度的了解。
  3. 針對建築物常見糾紛具有認識能力及實務處理能力。
  4. 掌握土地利用管制與建築管制的關係。
