##  課程簡介Course Description
```
課程介紹
幾點構想，以之為課程介紹：
一、已有個經、總經、福經、租稅理論、公共支出……等理論課程，既然有門「財政政策」，就主談政策。
二、18 周的課程，莫約四個月的時間，分成二部分。第一部分，以課程講述為主、討論為輔的方式進行。分成以下四個主題：
（一）、財政政策的沿革與發展；
（二）、賦稅改革的新趨勢；
（三）、當下稅改議題介紹；
（四）、其他財政政策議題介紹。
部分主題或洽詢與實際政策決策相關人士擔任講師。
三、課程的第二部分將以報告為主、討論為輔的方式進行。對於教育的想法，我念茲在茲的是—培育具有完整社會化人格的人，而高等教育則在於精雕細琢這格人格的中心思想。此一中心思想必須具備有：
（一）、獨立思考的能力 (independent thinking)；
（二）、邏輯與理性辯證的能力 (logical reasoning)；
（三）、預見、面對以及解決問題的能力 (problem solving)。
我期許本課程第二部分的議題討論，對於此三項能力的養成，能夠扮演積極的催化作用。
四、討論的議題由所有修課同學與老師共同提出後，每位修課同學隨機指定獨力研究議題。每個議題需進行至少兩次報告；每位同學研究議題數與每個研究議題報告次數視修課學生人數而定。
```
